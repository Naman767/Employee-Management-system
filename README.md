# Employee-Management-system
The Employee Management System is a desktop application developed in Java using Swing and AWT for the graphical user interface. It serves as a comprehensive tool for managing employee data effectively, allowing users to perform various operations such as adding, editing, deleting, and searching for employee records. The application utilizes MySQL as its database backend, connected via JDBC.
KEY FEATURES->
Add Employee: Easily add new employee records to the database.
Edit Employee: Update existing employee details with new information.
Delete Employee: Remove employee records from the database when necessary.
Search Employee: Find employees by name, department, or ID.
Display Employees: View all employee records in a structured table format.
Date Picker: Utilize JCalendar for selecting dates during data entry.
Database Connectivity: Connect seamlessly to a MySQL database using JDBC.
TECHNOLOGIES USED->
Java Swing: For creating the graphical user interface.
Java AWT: For event handling and additional GUI components.
JDBC: For database connectivity with MySQL.
MySQL Database: For storing employee data.
JCalendar: For date selection in forms.
